### 회원가입 방법
1. [생물안전 국가관리 통합관리시스템](http://125.6.37.38/mainmenu.do/)으로 이동합니다. 
2. 좌상단 로그인 버튼 아래에 있는 회원가입 버튼을 클릭합니다.
